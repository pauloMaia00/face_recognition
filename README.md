# Face Recognition Project

A Python-based face recognition application that detects and recognizes human faces by comparing them against a set of known identities. This project is designed for learning and experimentation with computer vision, face detection, and face embedding workflows.

> ⚠️ **Disclaimer:** This project is intended for educational purposes only. Be mindful of privacy, consent, and applicable laws when working with facial recognition technology.

---

## Features

- Build a dataset of **known faces**
- Generate and store **face encodings**
- Recognize faces in:
  - Images
  - Video files
  - Live webcam feed (optional)
- Draw bounding boxes and label recognized faces

---

## Tech Stack

- **Python 3**
- **OpenCV** – image and video processing
- **face_recognition** – face detection and encoding (built on dlib)
